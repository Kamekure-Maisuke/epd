# Elysia + pglite + drizzle

```bash
# パッケージインストール
bun install

# マイグレーションファイルの作成
bun run drizzle:generate

# マイグレーションファイルの適用
bun run drizzle:push

# 初期データの投入
bun run drizzle:seed

# 開発サーバーの起動
bun run dev

# drizzle studioの起動
bun run drizzle:studio
```
